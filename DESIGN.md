# VECT Design System v1.0

## 🎨 Visual Identity
- **Primary Focus**: Dark Tech / Minimalist Professional.
- **Core Colors**:
  - `background`: `#0A0A0A` (Deepest Black)
  - `surface`: `#111111` (Matte Graphite)
  - `accent`: `#FF6B1A` (Vector Orange)
  - `accent-bright`: `#FF8A4C` (Orange Highlight)
  - `data-success`: `#00C853` (Emerald Signal)
  - `data-error`: `#FF1744` (Alert Red)

## ✍️ Typography
- **Primary UI**: `Inter`, San-serif. Focus on readability and hierarchy.
- **Data/Financial**: `JetBrains Mono` or `Fira Code`. Used for all numerical values, tables, and scanning logs.

## 📐 Layout & Components
- **Spacing**: 8px (base unit). Containers usually `max-w-6xl`.
- **Borders**: Subtle `1px solid rgba(255,107,26,0.1)` for cards.
- **Micro-effects**:
  - **Glassmorphism**: 20px blur for modals and navigation.
  - **Glow**: Subtle orange glow for primary actions and active scanning states.
  - **Transitions**: 300ms ease-in-out for all hover states.

## 🤖 AI Interaction Guidelines (Radar)
- Real-time scanning should be represented with a pulsing "Signal Orange" dot.
- AI-extracted data should be distinguished with a specific "AI Badge" in cyan.
